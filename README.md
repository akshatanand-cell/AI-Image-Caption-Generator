An AI application that generates descriptive captions
for any uploaded image using Computer Vision.

## ✨ Features

- 🖼️ **Upload any image** — JPG, PNG, WEBP
- 🤖 **AI generates caption** instantly
- 👁️ **Computer Vision** — AI can actually SEE images
- 📊 **Word & character count** of generated caption
- 🎨 **Beautiful animated UI** with gradient background
- ⚡ **Fast results** powered by BLIP model

---

## 🧠 How The AI Works
User uploads image
⬇️
BLIP AI analyzes pixels
⬇️
Identifies objects, scenes, actions
⬇️
Generates natural language caption

## 🚀 How to Run

### Step 1 — Clone the repository
```bash
git clone https://github.com/akshatanand-cell/AI-Image-Caption-Generator.git
```

### Step 2 — Install dependencies
```bash
pip install streamlit transformers torch Pillow
```

### Step 3 — Run the app
```bash
python -m streamlit run app.py
```

### Step 4 — Open browser
http://localhost:8501

## 🎯 About BLIP Model

BLIP (Bootstrapping Language-Image Pre-training) is a
state-of-the-art Computer Vision model by Salesforce.
It's trained on millions of image-text pairs and can
understand and describe any image in natural language.
